# Webpack Starter Kit

# -------------------------

# Installation:
### 1. Clone the repository and install dependencies:
  ``` npm install ```

# Scripts: 
### Build:
  ``` npm run build ```
### Development Server:
  ``` npm run dev ```

Check out the live demo here:

- You can customize Webpack configuration in webpack.config.js.
