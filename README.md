# Webpack Starter Kit

# -------------------------

# Installation:
### 1. npm install
  - Download all dependencies
### 2. npm start build
  - Creates dist folder and files

# Commands: 
### 1. npm start build:
  - Saves edits in the "src" to "dist"
### 2. npm start dev:
  - run webpack server
